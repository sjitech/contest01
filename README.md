# 第一回の社内プログラミングコンテスト

## 利用方法
[Gradle](http://www.gradle.org)を使用します｡Eclipseでは､gradleのpluginをインストールし､gradleプロジェクトをインポートします｡

Ideaなら､gradleを既にサポートしますので､gradleプロジェクトをインポートします｡

核問題の確認はgradleでJunitコードを実行してください｡

問題5はMarkDownのファイルで解答します｡
